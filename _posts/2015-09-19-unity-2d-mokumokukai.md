# vagrantboxの紹介

  - Macでも簡単にvirtual boxのように仮想サーバー立ち上げる
  - [リンク](www.vagrantbox.es)
  - 使い方

```
$ vagrant box add {title} {url}
$ vagrant init {title}
$ vagrant up
```
   - メリット：設定などは一つのファイルで管理
   - folderを指定するとOS間のファイル共有もできる（便利ですね！）
