# 1. How to write switch in ios

* For integer

```
NSInteger n = 1;
switch (n) {
    case 1:
        NSLog(@"n is 1");
        break; //ここのbreakをはずした場合の動作
    case 2:
        NSLog(@"n is 2");
        break;
    default:
        NSLog(@"default");
        break;
}

//上記はこれを書いた結果と同じ
if(n == 1){
    NSLog(@"n is 1");
}else if(n == 2){
    NSLog(@"n is 2");
}else{
    NSLog(@"default");
}

```

* For string

There is no support for String so we can write like this.

```
NSString *lookup = @"Hearts"; // The value you want to switch on

typedef void (^CaseBlock)();

// Squint and this looks like a proper switch block!
// New ObjC syntax makes the NSDictionary creation cleaner.
NSDictionary *d = @{
    @"Diamonds":
    ^{
        NSLog(@"Riches!");
    },
    @"Hearts":
    ^{
        self.hearts++;
        NSLog(@"Hearts!");
    },
    @"Clubs":
    ^{
        NSLog(@"Clubs");
    },
    @"Spades":
    ^{
        NSLog(@"Spades");
    }
};

((CaseBlock)d[lookup])(); // invoke the correct block of code
```
